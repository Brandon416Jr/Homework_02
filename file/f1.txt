HTML

    超文本標記語言 (HyperText Markup Language)
    標記語言（markup language）
    Web 瀏覽器（Chrome、Edge、Firefox、Safari）會讀取 HTML 文檔並正確顯示它們。


"標籤"基本的架構：

    起始標籤 （The opening tag）：大於、小於的符號「< >」，裡面再放入元素名稱。
    結束標籤 （The closing tag）： 元素名稱前面多了個前置斜線「/」。代表這個元素的尾端。
    內容（The content）： 這個元素的內容。
    元素（The element）： 由起始標籤、結束標籤、內容所組成。

空標籤(empty elements) : 沒頭沒尾沒內容
e.g. <br>、<hr>、<input>、<img>

元素還可以有「屬性（Attribute）」

    每個標籤都可以有屬性
    利用屬性設定可以觸發這個標籤的主要技能，像是圖片(img標籤)可以透過src的屬性去幫我們給予圖片的路徑(來源)，讓瀏覽器可以顯示到頁面上
